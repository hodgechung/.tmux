# 🚀 快速参考卡

## 核心快捷键（记住这 5 个）

```
Ctrl+B, ?   → which-key 菜单 ⭐
Ctrl+B, F         → 模糊搜索
Ctrl+B, Shift+O   → 会话管理  
Ctrl+B, Space     → 快速复制
Ctrl+B, /         → 内容搜索
```

## 💡 记忆技巧

| 键 | 含义 | 记忆方法 |
|----|------|----------|
| **?** | Which-key | **W** = **W**hich 哪个键 |
| **F** | Fuzzy search | **F**uzzy 模糊搜索 |
| **Shift+O** | Overview | **O**verview 会话总览 |
| **Space** | Text thumbs | 空格 = 快速标记 |
| **/** | Search | 搜索符号 |

## 🎯 使用场景

### 不知道按什么？
→ `Ctrl+B, ?` 打开 which-key 菜单

### 找窗口/会话？
→ `Ctrl+B, F` 模糊搜索

### 管理多个会话？
→ `Ctrl+B, Shift+O` 会话管理器

### 复制屏幕文本？
→ `Ctrl+B, Space` 快速复制

### 搜索历史输出？
→ `Ctrl+B, /` 内容搜索

## 📊 状态栏

```
⌨ | CPU: 35.2% | 16:42 | 02-Apr | user@host
```

- **⌨** - Prefix 状态（按下 Ctrl+B 显示 ON）
- **CPU** - 实时使用率
- **时间** - 当前时间
- **日期** - 当前日期
- **用户** - 当前用户@主机名
